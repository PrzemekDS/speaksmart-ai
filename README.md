# 🗣️ SpeakSmart AI

**SpeakSmart AI** to interaktywna aplikacja językowa stworzona w Pythonie z wykorzystaniem **Streamlit** i **OpenAI GPT-4o-mini**.  
Pozwala tłumaczyć tekst w czasie rzeczywistym, śledzić historię tłumaczeń, budować własny słownik oraz kontrolować koszty użycia modelu.

---

## 🚀 Funkcje aplikacji

✅ **Tłumaczenie tekstu** — szybkie i precyzyjne tłumaczenia dowolnych słów lub zdań.  
✅ **Wykrywanie języka** — aplikacja automatycznie rozpoznaje język tekstu wejściowego.  
✅ **Historia tłumaczeń** — możliwość przeglądania ostatnich zapytań.  
✅ **Mój słownik** — zapisz przetłumaczone słowa do swojego słownika i eksportuj do pliku CSV.  
✅ **Statystyki i koszty** — aplikacja wyświetla liczbę tokenów i szacowany koszt tłumaczeń.  
✅ **Nowoczesny interfejs** — czytelny i prosty w obsłudze layout Streamlit.
